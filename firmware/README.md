# Firmware Layout

- `FWV2\eink_viet_custom_fw`: buildable DA14585 source tree.
- `FWV2\eink_viet_custom_pack_v2`: small packaged `EINK-VIET` image and README.
- `FWV2\pad_to_256kb.ps1`: helper to pad generated images to 256KB with `0xFF`.

Do not commit board-specific backups or donor dumps here. Keep those in the local lab workspace.
